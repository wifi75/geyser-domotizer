I terminali Vin+ / Vin- (a vite) non si collegano all'ESP32: vanno inseriti in serie su uno dei due fili che alimentano la pompa, cosi' la sua corrente attraversa fisicamente il modulo.

Aggiornamento manuale

        
Carica un file .bin compilato in locale (deve corrispondere esattamente alla scheda in uso).

        

          Scarica i file dalla pagina
          Release più recente su GitHub,
          scegliendo quelli per la tua scheda (es. esp32dev o xiao-esp32c3):
        

        
    
          
  1. firmware-<scheda>.bin — obbligatorio, caricalo per primo. Il dispositivo si riavvia da solo al termine.
    2. 
          
  2. littlefs-<scheda>.bin — solo se è cambiata anche l'interfaccia web (quasi sempre sì). Carica questo dopo, quando il dispositivo è di nuovo online: ogni upload sostituisce solo un file alla volta.
